从源代码编译安装——CMake 1. 前言
  此文档将介绍使用CMake编译安装cmdtr的方法，在开始前，请确保你的系统中安装了gcc和CMake。
2. 编译
  首先，进入源代码的根目录，新建一个build目录，然后进入。
  进入build目录后，键入如下命令： 
cmake ..
  随后CMake将开始检查和配置，如果没有问题，将会当前目录下生成一个Makefile文件。如果你不想让make命令使用默认的编译器配置，你可以使用-G选项指定。在生成Makefile后在命令行中键入make命令对源代码进行编译，编译通过后，生成的二进制文件将会保存在源代码根目录下的bin目录中。
