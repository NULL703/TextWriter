选项列表 表格版本：V11x-1
适用的程序版本：V1.1.x
下面列出的是当前版本可用的选项以及相关的简短说明，单击第一列的选项名可查看详细的使用方法。
cmdtr选项列表
选项名称 作用 类型 参数数量 参数类型 位置要求 其他要求
--text
-t		 以文本形式保存输入的内容 主选项 1 字符串 无 参数应为文件名或路径
--ascii
-a		 以数字序列形式保存输入的内容 主选项 1 字符串 无 参数应为文件名或路径
--read-as-ascii 显示数字序列文件的原始内容 主选项 ≥1 字符串 无 参数应为文件名或路径
--read-as-text 显示文本文件的内容 主选项 ≥1 字符串 无 参数应为文件名或路径
--showall
-A		 显示文件的全部内容 主选项 ≥1 字符串 无 参数应为文件名或路径，且不会对数字序列文件进行转码
--export-as-nse 转换为数字序列文件 主选项 1 字符串 无 参数应为文件名或路径
--export-as-ori 还原数字序列文件 主选项 2 字符串 无 参数应为文件名或路径。在批处理模式中只需要一个参数，该参数应当是目录的路径
--file2C 将普通文件转换为C源文件 主选项 1 字符串 无 参数应为文件名或路径
--nse2C 将数字序列文件转换为C源文件 主选项 1 字符串 无 参数应为文件名或路径
--help
-h		 显示帮助信息 特殊 0 无 应当单独出现 无
--bufsize 设置读取文件时所要显示的字符数，如果设为0，则一次性显示所有内容（默认值为1000） 附加选项 1 整数 无 无
--unlimited 取消文件转换的最大限制 附加选项 0 无 无 应当与--export-as-nse配合使用。如果使用了批处理选项，则此选项应在主选项之前
--continue
-c		 文件续写 附加选项 0 无 应当在写入选项之前 无
--batch
-b		 批处理选项 附加选项 0 无 应当在转换选项之前 此时的转换选项参数应指定为目录或目录的路径
--showline
-l		 查看文件内容时显示行号 附加选项 0 无 应当在查看选项之前，且应当是在最前面出现 只有在bufsize为0的情况下才会生效
--export-as-hex
-H		 以十六进制形式转储文件 主选项 1 字符串 无 参数应为文件名或路径
--dehex
-d		 还原已转储的HEX文件 主选项 2 字符串 无 参数应为文件名或路径。在批处理模式中只需要一个参数，该参数应当是目录的路径
