安装现有的二进制文件 注：本文档将侧重于介绍Linux下的安装方法。
1. 获取二进制文件
  在https://github.com/NULL703/TextWriter/releases可以找到最新版本或旧版本的发布内容，其中包括源代码和二进制文件。在选择二进制文件时，请选择与你的操作系统对应的版本；如果要从源代码开始编译，请选择cmdtr-src-full-[版本号].tar.gz。请勿直接克隆存储库，否则编译将无法通过！
2. 安装与配置
  在cmdtr的二进制程序包中包含有两个文件，一个是主程序，另一个则是相关的依赖库。在Windows下可以将其放在同一个目录中，但在Linux下需要将其放在系统指定的目录下，或者设置环境变量，由于前者的操作方法较为简单，且易于理解，所以本文档将采用第一种方法进行安装。
  操作方法很简单，首先，解压缩程序包中的文件，然后以管理员身份将其复制到指定的位置： 
cp cmdtr /bin
cp libformula.so /lib
注意：
  在红帽系的发行版中，libformula的存放位置应为/lib64或/usr/lib64，在文件复制完成后请执行ldconfig命令，以确保动态链接库能被正确调用。
  在完成以上操作后，即可在任意目录下使用此命令。在Windows下，如果要得到同样的效果需要将二进制文件所在的位置加入PATH环境变量，具体方法请自行查找。
